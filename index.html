Makes taking the bus feel like flying

Cambion is a light-weight open-source service bus for .NET with a set of convenience functions for synchronous and asynchronous event handling.

Get started

  • Lightweight

    With less than 400 lines of executable code in its core functionality, Cambion is a very lightweight component

  • Fast

    Events are handled in the single digit microsecond range *

  • Extensible

    Cambion can easily be expanded to use other network transports or data serializers

* Benchmarked using the default loopback without network communicaation

Support

In case you encounter some issues with Cambion you can use GitHub's issue tracker to see if your issue is a known issue (with possible workarounds), or you can create a new issue. Please be verbose when creating new issues to help with troubleshooting and debugging.

For more generic service bus questions please use the appropriate support services for the service bus supplier, or ask a question on StackOverflow. For Cambion related questions on StackOverflow, please use the "Cambion" tag if possible.
